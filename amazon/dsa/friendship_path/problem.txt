Consider a popular social networking website, Facebook for example.
People can create their own profile and add the profiles of others to their friends list.

You are given two Facebook profiles, "Gordon" and "Stephen".
I would like to find out how to get from Gordon's profile to Stephen's profile by navigating through Gordon's friends,
then friends of friends and so on. For example:
    * Gordon
    * Shantel (who is friends with Gordon)
    * Maria (who is friends with Shantel)
    * David (who is friends with Maria)
    * Stephen (who is friends with David)

I would like you to create an algorithm to automatically generate a list of profiles,
each of who is a friend of the previous entry, which link the starting profile to
the end profile. The algorithm will return the list of selected
profiles.
